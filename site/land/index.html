Eh Fake?

Classifique notícias como informativas ou desinformativas de forma simples e eficiente

O que o "Eh Fake?" faz por você?

1. Classifica automaticamente notícias em informativas, desinformativas ou neutras utilizando Inteligência Artificial.

2. Oferece uma interface simples para consulta de informações de forma transparente.

3. Integração completa com banco de dados e repositórios para auditoria dos resultados.
